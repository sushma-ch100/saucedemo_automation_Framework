# SauceDemo Automation Framework

## Overview
This project is a Selenium-based automation framework built using Java, TestNG, and Maven.  
It automates key user flows of the SauceDemo e-commerce application.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- ChromeDriver

## Project Structure
src/test/java/com/qa
- base → WebDriver setup
- pages → Page Object Model classes
- tests → Test classes

## Test Scenarios Covered
- Login functionality
- Add to Cart
- Checkout flow

## How to Run
1. Clone the repository
2. Navigate to project folder
3. Run:
   mvn clean test

## Features
- Page Object Model (POM)
- Explicit waits
- Cross-browser ready structure
- Clean and scalable framework

## Test Reports & Artifacts
- Test execution reports are available under `artifacts/reports`
- Failure screenshots are saved under `artifacts/screenshots`
